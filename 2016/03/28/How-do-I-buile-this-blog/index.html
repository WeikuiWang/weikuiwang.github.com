How do I buile this blog

发表于 2016-03-28 更新于 2016-04-01 分类于 Programming

最近一段时间的工作学习之余，我寻着互联网上点点信息，决定利用Github Pages + Hexo构建一个个人博客站点，以后的话希望生活工作学习中的点点滴滴都能有心给记录下来，那么，作为第一篇非功能性测试文章，一定得先记录下最近为搭建本博客所做事迹。之前的话从零开始时候就把操作流程记录到了我的CSDN上，所以，这里的话就只列出标题和链接就足够了。

  1. 使用GitHub pages + Hexo 快速搭建个性化博客(一)-部署Hexo默认blog模板至远程GitHub

  2. 使用GitHub Pages + Hexo 快速搭建个性化博客(二)-购买并解析域名

  3. 使用GitHub Pages + Hexo 快速搭建个性化博客(三)-更换博客主题

  4. 使用GitHub Pages + Hexo 快速搭建个性化博客(四)-博客菜单和侧边栏优化

  5. 使用GitHub Pages + Hexo 快速搭建个性化博客(五)-首页优化和文章分享评论

  6. 使用GitHub Pages + Hexo 快速搭建个性化博客(六)-站点SEO优化

    我在csdn浏览这几篇记录时发现当时排版并不是太美观，有图片还超出了页面中间的显示区域，但是内容绝对实时反映了我操作过程的 真实记录，而且有些过程如域名的购买和一些调试都是不能二次试验的，可谓是实况转播。

    当然了，期间很多过程我也都是参考官方文档和网上博友们的经验分享，最初我还犹豫自己做就做了，但要不要再来记录一番呢，最后结果还是得给记录下来，因为大家每个人在调试的过程中使用的机器和软硬件环境是不同的，这么多流程大家肯定会遇到官方文档或是网友们没有遇到过的情况，这样自己把问题给记录下来，解决后把方法也分享上去，最后的话大家就又有了可以参考的地方。而且后期我自己也是需要更换办公室或是自己电脑的，同样的操作也会遇到问题，回过头来查看自己的记录，能够更好的提高自己效率，也许这就是当下互联网发展中知识分享的精神之一吧。

    哈哈哈，说的好像很严肃一样，其实整个探索的过程回头来看还是很轻松愉快的，自己很主动的去改进每一处功能，在网上见到博友们有更好的内容和效果，自己也就想着研究一下，在此真的是很感谢前人们为我们后来着趟出一条康庄大道，要不我们现在遇到问题在搜索引擎中搜索时基本都能找到曾经遇到同样问题的人或神呢？
