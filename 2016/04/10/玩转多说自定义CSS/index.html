玩转多说自定义CSS

Posted on 2016-04-10   |   In Programming   |     |   Views


评论系统有几个比较有名的，既然我选择了多说，就一直把这个给用下去并尽量的优化，相信每个团队都会有用心经营自己的平台，包括多说。

安装上多说评论之后还没怎么精心设置过，这两天就琢磨着优化一下，慢慢的有发现再改进嘛。稍微搜索了一下资料，网上方法还是挺多的，当然还是主要依靠官方给出的CSS样式设置，从这里我们能了解到我们能修改到哪里。然而所谓官方就是比较正经，不符合每个人个性化需求，所以就有许多网友个性化的定制出不同系列的评论样式风格，在他们站点上看到的样式总感觉是那么和谐。多说官网里也给出了优秀的个例。我大致了解了一下能修改哪些区域，之后给我的评论系统添加了一系列的样式，给出相关CSS如下：

之前有看到网友说多说评论系统加载速度较慢，有时图片或文字显示不恰当等问题，我这个站点刚建好加载上多说时候并未感觉到，最近的话文章也有两页了，在不同电脑手机等终端也都有测试查看，确实发现些问题，之后的话就查找下原因，能有优化方法的话也跟大家分享出来。
