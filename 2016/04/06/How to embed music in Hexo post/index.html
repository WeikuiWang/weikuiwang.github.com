How to embed music in Hexo post

Posted on 2016-04-06   |   In Programming   |     |   Views


在单篇博文中添加音乐

目前为止我在Hexo官网插件中发现了三款用于在博客文章中添加音乐的插件，但使用方法看完之后，发现都有不满意的地方，要么是音乐来源问题，要不是使用方法、界面效果等。最后还是觉得在官网之外找下资源，Google一下果然找到一款安装使用简洁的插件：hexo-tag-netease-music。快速测试使用效果还是可以的，而且音乐来源于网易。可见我上一篇的推荐音乐的文章。
搬过来的使用方法：
Installation
To install, run the following command in the root directory of hexo:

npm install hexo-tag-netease-music –save

Add this plugin in your global configure file _config.yml.

plugins: hexo-tag-netease-music

Size
Size of iframe widget:
width range: [260, 510]
height range：[190, 500]
Size of flash widget:
small: {width:278, height: 32}
medium: {width:320, height: 66}

我试了下还是就使用iframe就行了，flash还要考虑浏览器插件和版本的问题(比如Safari就不支持flash)，而且外观不简洁。但我发现使用iframe时在pc上预览效果，播放器的下方还有200px左右高度的空白，而且设置的Iframe宽高度都没生效，暂时还未在css或js文件中修改成功，但初步效果是已经达到了。之后有小伙伴使用更好的官方或非官方音乐插件时或者是修改使用方式等还请推荐一下。
