Learn Python the hard way in three days

Posted on 2016-04-04   |   In Programming   |     |   Views


这次清明节放假三天时间里，由于是下雨天气，也没太好的理由出去感受美好春光，就定个短期目标吧。学习python有一段日子了，而之前买的所谓经典入门书籍Learn Python the Hard Way还在我书桌寂寞的躺着，为了巩固一下基础决定这几天就再次动手实践一下上面的内容。

之前刚买回来时候就完整的翻阅过几遍吧，感觉确实是入门级的书籍，总共52个代码段例子。很佩服作者Zed A.Shaw的耐心程度，真的是把新手级别的编程人员的问题都给总结了起来，甚至很多地方讲解了某个计算机操作技能，如某个字符怎么在键盘上敲出…以至于我最初只是翻阅，并未实践。
正所谓“纸上得来终觉浅，绝知此事要躬行”，果然如此，在第一天里，我完成了前面20个例子。我是在CentOS系统终端中练习这些例子，同时还查阅书中出现的不认识的单词，记录在手机的有道词典单词本里。在买这本书之前，网上参考了很多网友意见，很多过来人都有推荐这本书的，当然也有反面评价的，毕竟不同人有不同理解嘛。今天自己亲自动手练习下来的话，给出的还是正面的印象的，或许书中很多练习确实有啰嗦重复，但感觉很语重心长的语气有没有，所谓啰嗦只有家人和老师能给你带来这种感觉吧，这也正是对你真正关心对你负责的态度。作者作为资深程序猿要从这么基础的角度一个个解释这些内容，是需要很大耐心的，换我而言很多时候给人解释问题就忽略了基础背景的考虑。由此而言，我跟着Zed不只在学python，还在学着英语，还有对人对知识的态度。这几天给专门攻克下来这本书之后，确实又收获不少，包括python入门编程和编程之外的一些东西。
刚好今天，也就是在我练习的第三天休息翻手机时间，在简书上看到一个作者也是正在记录学习这本书的亲身历程，他比我记录的还要详细，每个例子作为一片文章记录到简书上面了，包括书中例子要求和代码内容还有自己的调试过程中出现的各种情况。他记录到了第26节，对于这位的练习和记录精神，我也得赞一个！看起来有些时候不认识的人也在你不曾去过的地方做着和自己一样的事。想想也是奇妙呢。
